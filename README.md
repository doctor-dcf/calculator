# 数学表达式计算器  
## 1.功能列表  
### 核心计算功能  
- 支持基础四则运算：+（加）、-（减）、*（乘）、/（除）  
- 支持取模运算：%（兼容浮点数取模，如 5.5%2.0=1.5）  
- 支持幂运算：^ 或 **（如 2^3=8、2**3=8，右结合性：2^3^2=2^(3^2)=512）  
- 支持括号优先级：()（如 (1+2)*3=9)  
- 支持正负号前缀：如 -5+3、(+4)*2、1*-3  
### 输入兼容与体验  
- 忽略输入中的空格：支持松散输入（如 1 + 2 * 3 与 1+2*3 效果一致）  
- 支持小数计算：如 3.14+2.86=6.0  
### 错误处理
- 非法字符检测（如字母、特殊符号 #/$ 等）  
- 括号不匹配校验（左 / 右括号数量不一致、缺少括号等）  
- 除数 / 取模为 0 拦截  
- 表达式格式错误检测（如运算符缺少操作数 1+、操作数多余 1 2 + 3 等）  
- 数字格式非法检测（如 12a3、12.34.56 等）  
## 2.编译 / 运行说明  
- 操作系统：Windows  
- 编译器：支持 C++11 及以上标准的编译器（如 GCC、Clang、MSVC）
## 3.核心算法说明
程序核心采用调度场算法（Shunting-yard Algorithm），分为三步完成计算：
1.词法分析（Tokenize）：将输入字符串拆分为数字、运算符、括号等 Token 单元；
2.中缀转后缀（Infix to Postfix）：通过运算符栈处理优先级和结合性，将中缀表达式转为后缀表达式（逆波兰式）；
3.后缀求值（Postfix Evaluation）：通过数字栈遍历后缀表达式，完成最终计算。
